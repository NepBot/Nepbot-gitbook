1. Create Off-chain Polls

Not everything happens in Discord needs to be on-chain.

Nepbot also allows you to create an off-chain poll right away, easy and quick.

To start an off-chain poll, all you need to do is

  1. Use the command /poll

  2. Enter the question first

    1. Press Enter if it’s a YES or NO poll

    2. Click on ‘+1 more’ to add multiple choices

  3. Others can directly react to vote for the options.
